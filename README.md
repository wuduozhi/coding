### 框架

* [iot-plat](https://gitee.com/pointiot/iot-plat)
    - 稳定、高并发，单服务器轻松支持8000+终端；
    - JAVA语言开发，可移植，支持Windows，Linux平台部署；
    - 采用优雅的松耦合架构，支持Spring；
    - 支持多编解码器；支持协议解析组件热插拔；
    - Web管理端支持多种报表展示方式。
    - 支持关系型数据库、内存数据库、实时数据库存储。

    - 开源实现，安全可控（欢迎加入开发）

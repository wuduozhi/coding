#### 0x04 添加传感器

* url:http://iot.wduozhi.xyz/api/sensor
* method:post

* description
    - 描述,string
* location
    - 地点,string
* factory
    - 厂商,string
* install_time
    - 安装时间,时间戳
* maintenance_time
    - 维修截止时间,时间戳
* produce_date:1234567890000
    - 生产时间,时间戳
* gate_id
    - 传感器id
* classify_id
    - 传感器类型id
